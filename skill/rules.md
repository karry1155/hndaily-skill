# 海南日报简报生成规则

## 1. 核心原则

1. **不复述标题**。价值在正文：具体生效日期、具体数字（GDP/增速/人数/金额）、具体人名（被查干部/获奖企业）、具体地点（XX镇/XX公司）、具体行为认定。
2. 抽取具体事实，丢弃官样套话（"深入贯彻""高度重视""扎实推进"）。
3. **按用户视角排序，不按版面顺序**：
   - ① 近期生效的新规、新政（影响办事/合规）
   - ② 反腐通报、干部被查
   - ③ 经济数据
   - ④ 节假日/出行/消费实用信息
   - ⑤ 重大事件收官/启动
   - ⑥ 中央层面动态
   - ⑦ 行业动态（仅当用户相关时）
4. **跨版合并同主题**：反腐横跨多版→合并一段讲；五一假期散在多版→合并讲。

## 2. 跳过或一笔带过

- 广告版（如第 011 版）
- 纯赛事流水（除非有破纪录或政策性意义）
- 健康科普通用知识（除非用户问健康）
- "专题"长篇政策解读（除非用户明确要细读）
- 农垦集团动态（除非用户做农业）

## 3. 数字/时间敏感

- 所有具体数字和日期保留原文
- "明起"换算成具体日期再写

## 4. 硬性条数上限

| 类别 | 上限 |
|------|------|
| 🔴 高优先级（近期生效政策/节前避坑） | ≤ 4 条 |
| 🟡 经济数据 | ≤ 3 条（同类合并，主指标+子指标写一行） |
| 🟢 重大事件 | ≤ 2 条 |
| 简讯 | ≤ 5 条 |
| **全文** | **≤ 600 字** |

## 5. 重大事件判定标准

按"一年后用户回头看，这件事是否还会被想起"判断：

- ✅ YES：破纪录、首次/末次、引发后续连锁政策、伤亡或重大事故、世界级赛事开闭幕
- ❌ NO（全部进简讯或删除）：开馆、剪彩、签约、会见、参观、调研、慰问、表彰、颁奖、揭牌、研讨会

反例：
- ❌ "海南科技馆开馆" → 开馆，进简讯
- ❌ "省委书记会见瑞士集团CEO" → 会见，删除
- ❌ "全省改革创新表彰大会" → 表彰，进简讯
- ✅ "亚沙会闭幕+攀岩世界纪录" → 破纪录，进重大
- ✅ "习近平就X在政治局集体学习讲话" → 中央层面信号，进重大

## 6. 反腐通报写法

不逐人列行为。按模板：
"{N} 起八项规定通报 + {M} 人接受审查调查；
覆盖 {系统列举}；
{N} 起均已开除党籍、移送司法。"

被查的人只列**姓名+职务**，不展开行为，除非用户后续追问。

## 7. 经济数据写法

同类数据合并为一行。主指标放前面，子指标跟在后面。
- ✅ 澄迈一季度 GDP 123.18 亿元、同比增长 4.2%，增速全省县域第一；固投 58.73 亿元（+19.3%）、规上工业增加值 12.76 亿元（+13.5%）
- ❌ 分三条写

## 8. 输出格式模板

```markdown
# 海南日报 {date}（{page_count} 版 / {article_count} 篇精读）

## 🔴 与你直接相关的（{子标题如"近期生效政策"或"节前避坑"}）
- 用粗体引导词 + 一句话事实 + 必要时附"信号/解读"

## 🟡 经济数据
- 列具体数字、增速、排名

## 🟡 {场景化主题，如"五一假期实用"}
- 路线/优惠/注意事项

## 🟢 重大事件
- 简短，每条 1–2 句

## 简讯
- 每条一行，可带版面号
```

字数：400–600 字。

## 9. 重写自查（输出前必过）

- [ ] 重大事件超过 2 条？砍到 2 条以内
- [ ] 重大事件里有"开馆/会见/签约/表彰"？挪到简讯
- [ ] 反腐部分超过 100 字？合并重写
- [ ] 经济数据超过 3 条？合并同主体数据为一行
- [ ] 任何一条简讯里能拆出"具体生效日期/数字/可避坑点"？提到🔴
- [ ] 全篇出现 ≥3 处"重要讲话""扎实推进""高度重视"？改掉
- [ ] 任何超过 30 字的句子能否换成"主语+数字+动作"短句？

自查找出不通过就重写，不输出不合格品。
